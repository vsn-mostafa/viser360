# SEO Images Folder

This folder contains Open Graph (OG) images for articles and pages.

## Usage

Place OG images here with descriptive names. Reference them in the article data using the `og_image` field.

Example: `/assets/images/seo/article-ai-future.jpg`

## Image Specifications

- **Recommended Size**: 1200x630px
- **Format**: JPG or PNG
- **File Size**: Keep under 1MB for optimal loading

## Naming Convention

Use descriptive, SEO-friendly names:
- `article-[title-slug].jpg`
- `page-[page-name].jpg`

Example:
- `article-ai-machine-learning.jpg`
- `page-about-us.jpg`
